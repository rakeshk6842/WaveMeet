## 1. Introduction

WaveMeet ("we," "us," "our," or "Company") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, disclose, and otherwise handle your information when you use our messaging application ("Service").

## 2. Information We Collect

### 2.1 Information You Provide Directly

**Account Registration:**
- Name
- Email address
- Phone number (optional)
- Profile picture
- Password (encrypted)
- Bio/Status message

**Communications:**
- Messages and chat content
- Files and media shared in conversations
- Voice and video call recordings (if enabled)
- Call duration and participants
- Typing indicators and read receipts

**Device Information:**
- Device type and model
- Operating system and version
- Device ID/UUID
- Push notification tokens
- WiFi/mobile network information

### 2.2 Information Collected Automatically

**Usage Data:**
- Login times and frequency
- Features used
- Search queries
- Conversation participants
- Last seen timestamps

**Technical Data:**
- IP address
- Browser/app version
- Crash reports and error logs
- Performance metrics
- Analytics data

**Location Data:**
- Approximate location (if permissions granted)
- Country and timezone

### 2.3 Information from Third Parties

- Data from social media platforms (if you sign in via social login)
- Contact list information (if permission granted)
- Information from chat participants

## 3. How We Use Your Information

### 3.1 Primary Uses

We use your information to:

- **Provide the Service:** Deliver messaging, calling, and chat features
- **Authentication:** Verify your identity and prevent unauthorized access
- **Communication:** Send you account, security, and service notifications
- **Improvements:** Analyze usage patterns and improve features
- **Support:** Provide customer service and technical support
- **Security:** Detect, prevent, and address fraud, abuse, and security incidents
- **Legal Compliance:** Comply with laws and respond to legal requests

### 3.2 Secondary Uses

- **Personalization:** Customize your experience
- **Marketing:** Send promotional content (with your consent)
- **Analytics:** Understand user behavior and trends
- **Research:** Conduct studies to improve our services

## 4. Data Storage and Security

### 4.1 Data Storage

- Messages are stored encrypted on our secure servers
- Media files are stored in encrypted cloud storage
- User profiles are stored in PostgreSQL database
- All data is backed up with encryption

### 4.2 Security Measures

We implement industry-standard security practices:

- **Encryption:** End-to-end encryption for messages (TLS 1.3)
- **Authentication:** JWT tokens with secure validation
- **Password Security:** bcryptjs hashing with salt rounds
- **HTTPS:** All communications use HTTPS
- **Firewalls:** Network-level protection and DDoS mitigation
- **Access Controls:** Role-based access control (RBAC)
- **Monitoring:** 24/7 security monitoring and intrusion detection
- **Compliance:** Regular security audits and penetration testing

## 7. Data Retention

**Active Accounts:**
- Messages: Retained until deleted by user or account deletion
- Media files: Retained until deleted by user
- Profile data: Retained while account is active

**Deleted Accounts:**
- Messages deleted
- Media files deleted
- Backup data deleted within 30 days
- Anonymized analytics data retained

**Business Records:**
- Transaction logs: 7 years
- Support tickets: 1 year
- Security logs: 90 days
